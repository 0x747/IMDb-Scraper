# IMDb Scraper [BETA]
A simple webscraper which can extract metadata from a movie or show's IMDb page

## Attributes

|      Name      |  Data type  |
|:--------------:|:-----------:|
| title          | str         |
| original_title | str         |
| title_type     | str         |
| year           | int         |
| end_year       | int         |
| day            | int         |
| month          | int         |
| date           | str         |
| runtime        | int         |
| age_rating     | str         |
| imdb_rating    | int         |
| votes          | int         |
| plot           | str         |
| poster_url     | str         |
| trailer_url    | str         |
| url            | str         |
| genre          | list        |
| cast           | list        |
| directors      | list        |
| writers        | list        |
| keywords       | list        |
